Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B que não é semântica

Nesta outra frase temos um termo em destaque usando a tag STRONG que é semântica.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I que não é semântica.

Nesta outra frase, temos um termo em ênfase usando a tag EM que é semântica.

Texto marcado

Podemos criar também um texto marcado com fundo esverdeado usando a tag MARK

O texto marcado também pode ser colorido usando a tag style na Head do HTML

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto inserido

Podemos, também, marcar como um texto inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U que não é semântica.

Texto sobrescrito

Pode ser usado para inserir coisas do tipo x20+3

Texto subscrito

Este pode ser usado para inserir coisas do tipo H2O
